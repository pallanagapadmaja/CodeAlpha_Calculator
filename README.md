# CodeAlpha_Calculator

## Overview
Calculator is a simple web-based application developed as part of the CodeAlpha Frontend Development Internship. It is designed to perform basic arithmetic operations through an interactive and user-friendly interface.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Clear Screen
- Real-time Result Display
- Responsive Design
- Interactive Buttons

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Structure
CodeAlpha_Calculator/
│── calculator.html  

## File Description

### calculator.html
Contains the complete calculator application including HTML structure, CSS styling, and JavaScript functionality in a single file. It manages layout design, button interactions, arithmetic operations, and result display.

It includes:

- Calculator heading/title
- Display screen for input and results
- Number buttons (0–9)
- Arithmetic operator buttons (+, -, ×, ÷)
- Equal button for calculation
- Clear button to reset screen
- Real-time input handling
- Responsive layout for different screen sizes
- Interactive button click functionality
- Styling enhancements for better user experience
- Keyboard support for easy operation

## Functionalities

### Number Input
Allows users to enter numeric values.

### Arithmetic Operations
Performs basic mathematical calculations.

### Display Screen
Shows entered values and calculated results.

### Clear Function
Resets the calculator screen.

### Real-Time Interaction
Processes user input instantly.

## How to Run the Project
1. Download the files  
2. Open calculator.html in any browser  
3. Start performing calculations
